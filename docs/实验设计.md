# 要从头到尾重新训练的实验计划：
分成各个层级，部分可以组合，excel各列是一层，没行是一个可能并且推荐实验的组合：
- a层：各个指引latent dynamics的类hnn/phnn/nueral ode的网络类型
- b层：ae训练相关的实验，应该是一个独立的层级，探究怎么让128维的latent representation更加适合hnn/phnn这种网络。（相对a层可能只对a层几个选择适用）
- c层：inference time的区别，有三种，第一种是像目前frameworka里的直接靠两次微调的lora进行推理。第二种在这基础上生成多个答案rerank。第三种是hnn rollout与lora推理结果在latent层面加权平均，也就是之前的framework b
- d层：应该是对hnn单独训练然后sft/和sft一起训练的对照


注意：experiment文件夹装wrapper，每个实验写清楚在每个层级里的设置，然后调用method文件夹里对应的训练/推理代码！
